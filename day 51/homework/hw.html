რა არის Dimensions და მისი ერთეულები?

Dimensions ნიშნავს ელემენტის ზომებს, ანუ რამდენად ფართო და მაღალია რაღაც ვებგვერდზე.

ძირითადი ზომებია:

  • width სიგანე
  • height სიმაღლე

ერთეულები, რომლებიც ზომებისთვის გამოიყენება:

  • px პიქსელი (ეკრანის პატარა წერტილი)
  • % პროცენტი (მშობელ ელემენტთან შედარებით)
  • em ელემენტის ტექსტის ზომაზეა დამოკიდებული
  • rem ძირითადი ტექსტის ზომაზეა დამოკიდებული
  • vw ეკრანის სიგანის პროცენტი
  • vh ეკრანის სიმაღლის პროცენტი

მაგალითი:

width: 100px;   /* სიგანე იქნება 100 პიქსელი */
height: 50%;    /* სიმაღლე იქნება მშობლის 50% */

Pixel და Responsive დიზაინი

6) რა არის Pixel?

Pixel (პიქსელი) არის ეკრანის პატარა წერტილი, რომელიც ფერს აჩვენებს.

ეს არის ვებსაიტების უმცირესი საზომი ერთეული.

ყველა მოწყობილობაზე ერთი და იგივე ზომის არ არის მაგალითად, ტელეფონზე პიქსელი პატარაა, კომპიუტერზე კი უფრო დიდი.

მაგრამ CSS-ში 1px ყოველთვის ნიშნავს ერთ ვიზუალურ ერთეულს, ზომა ავტომატურად ეგუება ეკრანს.

7) რა არის Responsive დიზაინი?

Responsive დიზაინი ნიშნავს, რომ ვებსაიტი ეკრანის ზომაზე ავტომატურად ერგება — ტელეფონზე, ტაბლეტზე და კომპიუტერზე კარგად ჩანს.

რატომ არის კარგი პროცენტების (%-ის) გამოყენება?

  • ელემენტების ზომა ერგება მშობელს ან ეკრანის ზომას
  • გვერდი უკეთ გამოიყურება ყველა მოწყობილობაზე
  • კოდიც უფრო მოქნილია

მაგალითი:

width: 50%;   /* სიგანე იქნება ეკრანის ან მშობლის ნახევარი */
